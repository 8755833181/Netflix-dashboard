# Netflix-dashboard
1) Project Title / Headline
   Netflix Catalog Explorer — Power BI Dashboard

2) Short Description / Purpose
   Interactive Power BI dashboard to explore Netflix titles by genre, country, year and rating.
   A visual analysis of Netflix’s library to find trends in content release and popularity.

3) Tech Stack
   The dashboard was built using the following tools and technologies.
   
   Power BI Desktop - Main tool used to create an interactive Netflix dashboard and visualize insights.
   Power Query - Used for data cleaning, transformation, and shaping raw data.
   DAX (Data Analysis Expressions) - Used to create calculated measures and columns for analyzing total shows, genres, ratings, and           release trends.
   File Format - .pbit for development and .png for dashboard previews.

4) Data Source
    The dataset used for this dashboard is the “Netflix Movies and TV Shows” dataset from Kaggle.
    Source: https://www.kaggle.com (search for “Netflix Movies and TV Shows”)
   
    It contains comprehensive information about all the shows and movies available on Netflix up to the data collection date.
    This dataset helps to analyze content trends, production patterns, and viewer preferences across different countries and genres.

5) Features and Highlights

    Country-wise Coverage
    Explore how many titles are available per country and compare regional catalogs using an interactive map and filters.

    Genre Breakdown
    See distribution across genres (Drama, Comedy, Documentary, etc.), and compare Movies vs TV Shows with stacked bar charts.

    Release Year Trends
    Track production volume over time — identify growth periods, drops, and recent trends in new releases.

    Rating & Duration Analysis
    Understand how viewer ratings and runtime (duration) vary by genre and content type.
